# How Network Conjestion Affects Keystroke Authentication Accuracy

## Overview
This project is part of my university dissertation. It investigates how network conditions such as **latency** and **jitter** affect the accuracy of **keystroke dynamics authentication**

The project uses a pre-existing keystroke authentication library('keystroke-dynamics') and integrates it into a mininet network simulation.

## Purpose

The goal of this project is to :

- Integrate a pre-existing keystroke authentication system into a networked environment
- Simulate network congestion and latency
- Measure and analyse how these conditions affect authentication accuracy
